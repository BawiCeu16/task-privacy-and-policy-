This Privacy Policy explains how Task App (the "App") handles your information. By using this App.

1. Open Source Information

This app is now open-source and free to use. The source code is available for review and contribution:

  • Complete transparency - anyone can inspect the code
  • Free to use with no hidden costs
  • Community contributions are welcome

2. No Data Collection

This App does not collect, store, or transmit any personal data to external servers. All your tasks and app-related data are stored only on your device using local storage (Shared Preferences).

3. No Ads or Tracking

  • The App does not display any advertisements
  • The App does not use any analytics or tracking tools
  • No data is shared with third-party companies or advertisers

4. Data Storage & Security

All your tasks and settings are saved locally on your device and are not backed up on any cloud service.

If you uninstall the App, all stored data will be permanently deleted.

Since the data is stored only on your device, we cannot recover lost data if the app is deleted.

5. App Permissions

The App does not require any special permissions (like camera, contacts, or location) as it only uses local storage to save your tasks.

6. Changes to This Policy

We may update this Privacy Policy in the future. Any changes will be posted here, so please check back periodically.
